 1. Procédure dot_product

 Le produit scalaire de deux vecteurs est : 

ps=v1[0]×v2[0]+v1[1]×v2[1]+...+v1[n−1]×v2[n−1]

 Pseudo-code (procédure)
Procédure dot_product(v1, v2, n, ps)
    ps ← 0

    Pour i de 0 à n-1 faire
        ps ← ps + v1[i] * v2[i]

    FinPour
FinProcédure

 Ici :

v1, v2 → tableaux (vecteurs)
ps → résultat (produit scalaire)
 2. Vérifier si deux vecteurs sont orthogonaux

 Deux vecteurs sont orthogonaux si :

produit scalaire = 0
 Algorithme avec la procédure
Début
    Lire n   // nombre de couples

    Pour k de 1 à n faire
        Lire taille

        Lire v1
        Lire v2

        Appeler dot_product(v1, v2, taille, ps)

        Si ps == 0 alors
            Afficher "Orthogonaux"
        Sinon
            Afficher "Non orthogonaux"
        FinSi
    FinPour
Fin

 3. Version avec fonction (au lieu de procédure)


 Fonction dot_product
Fonction dot_product(v1, v2, n)
    ps ← 0

    Pour i de 0 à n-1 faire
        ps ← ps + v1[i] * v2[i]
    FinPour

    Retourner ps
FinFonction
 Algorithme avec fonction
Début
    Lire n

    Pour k de 1 à n faire
        Lire taille
        Lire v1
        Lire v2

        ps ← dot_product(v1, v2, taille)

        Si ps == 0 alors
            Afficher "Orthogonaux"
        Sinon
            Afficher "Non orthogonaux"
        FinSi
    FinPour
Fin
